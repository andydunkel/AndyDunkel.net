Software

Die folgende Liste enthält nur Software welche ich privat, neben meinem Hauptberuf als Softwareentwickler, entwickelt habe.

DA-FormMaker

Der DA-FormMaker ist eine Software zur Erstellung von HTML-Formularen. In wenigen Minuten kann der Benutzer ein funktionsfähiges Formular für seine Homepage erstellen, ohne sich über HTML-Codes und Scripte Gedanken machen zu müssen. Die Software wird seid vielen Jahren aktiv weiterentwickelt und verfügt über eine breite Benutzerbasis.

DA-BestellFormular

Im Prinzip der DA-FormMaker mit der Spezialisierung zur Erstellung von Bestellformularen. Wer keinen kompletten Online-Shop betreiben und administrieren will, findet hier eine einfache Lösung seine Produkte über das Internet zu vertreiben.

Zur Hompepage

DA-QuizMaker

Eine Software zur Erstellung von Online-Quiz. Der Benutzer legt die Fragen und Antworten fest, verschiedene Fragen sind möglich, z.B. Single-Choice, Multiple-Choice oder freie Antworten. Auf Wunsch werden die Antworten und eine Auswertung angezeigt.

DA-CDStart

Die Software ist ein Nebenprodukt eines Problems entstanden: Wir wollten unsere Software auf CD vertreiben und mit einem CD-Menü ausstatten. Nachdem ich das CD-Menü programmiert hatte, habe ich die Software um eine grafische Konfigurationsoberfläche erweitert und heraus kam DA-CDStart. Dem Benutzer stehen 3 Menüsysteme zur Verfügung.

DA-HtAccessManager

Wer viele passwortgeschützte Benutzer mittels .htaccess verwalten möchte, findet hier eine einfache Lösung. Die Benutzer lassen sich einfach in der GUI anlegen, bearbeiten und löschen. Anschließend lädt man die neue .htaccess-Datei einfach auf seinen Webspace.
